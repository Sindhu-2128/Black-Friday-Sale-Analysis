# Black-Friday-Sale-Analysis
Analyzed Black Friday sales data using Python to understand customer purchasing behavior and product demand patterns. Performed data cleaning, exploratory data analysis, and visualizations to identify trends based on customer demographics and product categories. The project provides insights to support data-driven marketing and sales strategies.

## Project Overview

This project analyzes Black Friday sales data to understand customer purchasing behavior and identify key factors influencing purchase amounts. The analysis focuses on demographic attributes, product categories, and spending patterns to derive actionable business insights.

Objectives

- Analyze customer demographics (age, gender, marital status, occupation)

- Understand purchase behavior across product categories

- Identify patterns affecting purchase amount

- Prepare clean, structured data for analysis and modeling
